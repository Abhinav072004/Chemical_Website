# IIT Indore Chemical Engineering Department Website

A modern, responsive website for the Department of Chemical Engineering at Indian Institute of Technology Indore.

## 🚀 Features

- **Modern Design**: Clean, professional interface with smooth animations
- **Responsive Layout**: Optimized for all devices (mobile, tablet, desktop)
- **Academic Focus**: Comprehensive sections for programs, faculty, and research
- **Interactive Elements**: Hover effects, transitions, and micro-interactions
- **Performance Optimized**: Built with Vite for fast loading and development

## 🛠️ Tech Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Linting**: ESLint
- **Development**: Hot Module Replacement (HMR)

## 📦 Installation

1. Clone the repository:
```bash
git clone <your-repository-url>
cd iit-indore-chemical-engineering
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`


## 🚀 Deployment

### Netlify
1. Build the project: `npm run build`
2. Deploy the `dist` folder to Netlify
3. Configure custom domain if needed

### Vercel
1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect Vite and deploy

### GitHub Pages
1. Install gh-pages: `npm install --save-dev gh-pages`
2. Add deploy script to package.json:
```json
"scripts": {
  "deploy": "gh-pages -d dist"
}
```
3. Run: `npm run deploy`
